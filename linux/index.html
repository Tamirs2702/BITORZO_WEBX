Bitorzo for Debian-based Distros

curl -s https://updates.Bitorzo/desktop/apt/keys.asc | sudo apt-key add -
echo "deb [arch=amd64] https://updates.Bitorzo/desktop/apt xenial main" | sudo tee -a /etc/apt/sources.list.d/Bitorzo-xenial.list
sudo apt update && sudo apt install Bitorzo-desktop
To use the Bitorzo desktop app, Bitorzo must first be installed on your phone.
